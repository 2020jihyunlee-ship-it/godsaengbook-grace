# 갓생북 은혜 — 사용자 플로우 (User Flow)
> v1.0 · 2026.03 · PRD v1.0 기반

---

## 페르소나

- **계정 사용자**: 교회 리더, 교사, 간사. 이벤트 생성 및 관리. PC/모바일.
- **참여자**: QR로 접속하는 학생, 셀원, 선교팀 단원. 모바일 중심.

---

## A. 계정 사용자 플로우

### A-1. 회원가입 & 이메일 수집

1. 사용자가 갓생북 은혜 웹사이트에 접속한다.
2. 랜딩페이지에서 **예시 플립북 미리보기**를 확인하고 '무료로 시작하기' CTA를 클릭한다.
3. 회원가입 페이지로 이동한다.
   - 입력 항목: 이메일, 비밀번호, 이름(닉네임), 소속 교회명 (선택)
   - **마케팅 수신 동의 체크박스** (선택): "갓생북 은혜의 새 기능, 업데이트 소식을 받겠습니다."
   - 체크 여부와 관계없이 이메일은 수집됨 (서비스 공지 목적)
4. Supabase Auth로 계정 생성. `users` 테이블에 row 생성.
   - `marketing_consent`: true / false 저장
5. 로그인 성공 후 이벤트 생성 대시보드로 이동.

### A-2. 이벤트 생성

1. 대시보드에서 **'새 이벤트 만들기'** 버튼 클릭.
2. 이벤트 유형 선택: 수련회 / 선교여행 / 캠프 / 예배 / 모임
3. 이벤트 정보 입력:
   - 이름, 기간(시작일~종료일), 예상 참여 인원
   - 카테고리 선택 (이벤트 유형과 동일)
4. 시스템이 events 테이블에 저장 + **고유 QR 코드 URL 생성**.
5. 생성된 QR 코드가 화면에 표시됨. 리더가 참여자에게 공유.

> **개인 기록 이벤트**: 유형 선택 시 '개인 기록' 선택 가능. QR 코드 없이 본인이 직접 접속.

### A-3. 이벤트 관리 & 모니터링

1. 대시보드에서 이벤트 목록 확인.
2. 특정 이벤트 선택 → 상세 대시보드 진입.
   - 단체 이벤트: 참여자 목록, 참여자별 기록 수, 전체 기록 현황
   - 개인 이벤트: 본인 기록 수, 전체 기록 수
3. **그룹 공통 콘텐츠 업로드**: 단체 사진, 말씀 공지, 마무리 총평 텍스트 업로드.
   - 모든 참여자 플립북에 자동 삽입.

### A-4. PDF 다운로드 (무료, PC 전용)

1. 이벤트 대시보드에서 'PDF 다운로드' 버튼 클릭 (PC 브라우저 전용).
2. 다운로드 옵션 선택:
   - 개별 참여자 PDF (참여자 선택)
   - 그룹 통합 PDF (전체 참여자 + 그룹 콘텐츠)
3. PC 브라우저에서 `html2pdf.js`가 클라이언트 사이드로 PDF 빌드.
   - Supabase DB에서 entries, group_contents 전체 조회
   - 플립북 레이아웃 HTML 구성 → PDF 변환
4. 완료 시 PC에 `.pdf` 파일 다운로드.
   - 모바일 접속 시 "PC에서 다운로드해 주세요" 안내 표시.

---

## B. 참여자 플로우 (단체 이벤트) — 📱 모바일

### B-1. QR 접속 & 이름 입력

1. 참여자가 리더에게 받은 **QR 코드를 모바일로 스캔**.
2. 갓생북 은혜 웹앱의 해당 이벤트 접속 페이지로 이동.
3. 이름 입력 (소속/반/번호 선택 입력 가능 — 동명이인 구분용).
4. 시스템이 `participants` 테이블에 row 생성. 브라우저 `localStorage`에 세션 캐싱.
5. **나의 플립북 대시보드** 표시. 첫 방문 시 환영 메시지 + 기록 가이드 표시.

### B-2. 기록 입력

1. '기록하기' 버튼 탭.
2. 섹션 선택 (리더가 설정한 일정 기반 — 수동 선택).
3. 기록 입력 페이지:
   - **묵상 가이드 질문 표시**: 이벤트 유형 + 섹션에 맞는 질문 (예: "오늘 말씀 중 내 마음에 꽂힌 한 문장은?")
   - **예시 완성글 보기** 버튼: 실제 잘 쓴 예시글을 별도 UI로 표시
   - 텍스트 직접 작성 (최대 500자)
   - 성경 구절 직접 입력 필드 (선택) — placeholder: 예) "빌립보서 4:13"
   - 인용 문구 직접 입력 필드 (선택) — placeholder: 예) "오늘 강사님이 하신 말씀"
4. 사진 첨부:
   - 카메라 촬영 또는 사진첩 업로드 (권장)
   - 업로드 유도 안내: "📸 사진 한 장이 기억을 살려줘요!"
   - 사진 없을 경우: 기본 템플릿 이미지 선택 UI 표시 (필수 선택)
5. '저장하기' 버튼 탭.
6. 저장 완료 → 플립북 대시보드로 복귀, 새 페이지 실시간 반영.

> **기록 횟수 제한 없음** — 완전 무료.

### B-3. 플립북 감상

1. '나의 플립북' 대시보드에서 플립북 뷰어 진입.
2. react-pageflip이 클라이언트에서 렌더링.
   - 본인 기록 페이지 + 리더 업로드 그룹 콘텐츠 페이지 통합
3. 스와이프 또는 넘김 버튼으로 책장 넘기기.
4. 새 기록/콘텐츠 추가 시 "새로운 페이지가 추가되었습니다!" 알림.

---

## C. 개인 기록 플로우 (계정 사용자 본인)

1. 로그인 후 개인 이벤트 대시보드 진입.
2. 기록 입력 과정은 B-2와 동일.
3. 플립북 감상 및 PDF 다운로드 가능.

---

## 플로우 요약도

```
[랜딩페이지] 예시 보기 → '무료로 시작하기'
      │
      ▼
[회원가입] 이메일 + 마케팅 수신 동의 수집
      │
      ▼
[이벤트 생성] 유형 선택 → QR 코드 발급 (단체) / 직접 접속 (개인)
      │                          │
      ▼                          ▼
[모니터링 대시보드]        [참여자] QR 스캔 → 이름 입력
      │                          │
      ▼                          ▼
[그룹 콘텐츠 업로드]       [기록 입력] 직접 작성 + 사진 업로드
      │                          │
      └──────────┬────────────────┘
                 ▼
          [플립북 뷰어] 실시간 감상
                 │
                 ▼
          [PDF 다운로드] 무료, PC 전용
```
