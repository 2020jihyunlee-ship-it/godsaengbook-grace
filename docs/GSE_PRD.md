# [PRD] 갓생북 은혜 (God-Saeng Book Grace)
> v1.0 · 2026.03 · 교회 전용 완전 무료 버전

---

## 1. 제품 개요 (Product Overview)

### 비전
수련회, 선교여행, 캠프, 예배, 셀 모임 등 **교회 공동체의 소중한 경험**을 각자의 신앙 서사로 기록하고, 평생 소장 가능한 디지털 플립북 & PDF로 남깁니다.

### 핵심 가치
- **완전 무료**: AI API 없이 운영. Vercel + Supabase 무료 티어만으로 지속 가능.
- **교회 전용**: 성경 구절, 인용 문구 포함 모든 기록을 참여자가 직접 작성.
- **간편한 기록**: 사진 + 직접 작성. 묵상 가이드 템플릿으로 누구나 잘 쓸 수 있도록 안내.
- **모바일 최적화**: 참여자는 앱 설치 없이 QR 스캔만으로 기록 시작.

### AI 제거 — 완전 직접 작성 방식

| 기존 갓생북 (AI) | 갓생북 은혜 (직접 작성) |
|---|---|
| AI 에세이 자동 생성 | 본인이 직접 작성 (묵상 가이드 + 예시 템플릿 제공) |
| AI 성경 구절 추천 | 성경 구절 직접 입력 (선택 사항) |
| AI 인용 문구 추천 | 인용 문구 직접 입력 (선택 사항) |
| AI 이미지 생성 | 직접 사진 업로드 권장 / 없으면 기본 템플릿 이미지 선택 |

## 2. 대상 사용자

### 계정 사용자 (Account User)
- 교회 리더, 교사, 간사, 목사 등 **이벤트를 만들고 관리하는 사람**
- PC 또는 모바일 사용
- 회원가입 필수

### 참여자 (Participant)
- 수련회 학생, 선교팀 단원, 셀 구성원 등 **QR로 참여하는 사람**
- 모바일 중심
- 회원가입 불필요 (이름 입력만으로 접속)

### 개인 기록자
- 계정 사용자 본인이 개인 묵상/여정을 기록
- 회원가입 필수 (계정 사용자와 동일)

## 4. 핵심 기능

### ① 기록 입력 시스템
- **텍스트 직접 작성** (본문, 최대 500자)
- **성경 구절 직접 입력** (선택): 참여자가 마음에 와 닿은 구절을 직접 타이핑
- **인용 문구 직접 입력** (선택): 설교, 강의, 나눔 중 마음에 남은 말을 직접 입력
- **묵상 가이드 템플릿**: 이벤트 유형 + 섹션에 맞는 가이드 질문 + 완성 예시글 제공
  - 예) 수련회: "오늘 말씀 중 내 마음에 꽂힌 한 문장은? 왜 그 말씀이 나에게 왔을까?"
  - 예) 선교여행: "오늘 만난 사람이나 장면 중 하나님의 마음이 느껴진 순간은?"
  - 예) 셀 모임: "오늘 나눔에서 새롭게 깨달은 것, 또는 누군가의 이야기가 내게 전해준 것은?"
  - 예시 완성글: 실제 잘 쓴 글처럼 보이도록 placeholder가 아닌 별도 예시 UI로 제공

### ② 사진 업로드
- 카메라 직접 촬영 또는 사진첩에서 선택
- **업로드 적극 권장**: "사진 한 장이 기억을 살려줘요!" 안내 메시지
- **사진 없을 경우**: 기본 템플릿 이미지 선택 필수 (교회/자연/말씀 분위기 제공)
- Supabase Storage 저장

### ③ 플립북 뷰어 (모바일)
- `react-pageflip` 기반 클라이언트 렌더링
- 책장 넘기는 애니메이션 + 사운드
- 참여자 기록 페이지 + 리더 업로드 그룹 콘텐츠 페이지 통합 구성
- PWA 지원 (오프라인 캐시)

### ④ 랜딩페이지 (예시 보기 포함)
- 서비스 소개 + 이벤트 유형 안내
- **예시 보기 섹션**: 실제 플립북이 어떻게 만들어지는지 샘플 플립북 미리보기 제공
  - 수련회 예시 / 선교여행 예시 등 유형별 샘플
- 가입 유도 CTA

### ⑤ PDF 다운로드 (무료)
- 계정 사용자 PC 브라우저에서 클라이언트 사이드 빌드 (`html2pdf.js`)
- 서버 비용 없음
- 다운로드 옵션: 개별 참여자 PDF / 그룹 통합 PDF

### ⑥ 그룹 공통 콘텐츠 업로드
- 리더가 단체 사진, 말씀 공지, 마무리 총평 텍스트를 업로드
- 모든 참여자 플립북에 자동 삽입

### ⑦ 이벤트 모니터링
- 참여자 목록, 기록 현황 실시간 확인
- QR 코드 재공유

## 7. 기대 효과
- 교회 공동체가 비용 부담 없이 신앙 여정을 기록하고 소장
- 리더 입장: QR 하나로 전체 기록 자동 수집 + PDF 책자 완성
- 참여자 입장: 내 이야기가 예쁜 책이 되는 경험
- 직접 쓴 성경 구절과 인용 문구로 더욱 진정성 있는 신앙 기록
